# imbr-label

## 后端项目

## 推荐的IDE初始化

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (并禁用 Vetur).

## 自定义配置

参考 [Vite 配置选项](https://vitejs.dev/config/).

## 项目初始化

```sh
npm install
```

### 编译与热更新部署

```sh
npm run dev
```

### 编译并构建产品

```sh
npm run build
```
